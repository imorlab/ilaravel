## Acerca del Proyecto

Este es un sistema de productividad personal desarrollado con Laravel 10, que incluye varias herramientas para mejorar la gestión de tareas diarias y notas personales.

### Características Principales

#### Sistema de Notas
- Creación, edición y eliminación de notas
- Categorización (Trabajo, Personal, Ideas)
- Sistema de favoritos
- Búsqueda en tiempo real
- Interfaz moderna con diseño glassmorphism

#### Dashboard
- Vista general de estadísticas
- Gráficos de progreso
- Actividades recientes
- Calendario de actividades

### Tecnologías Utilizadas

- **Framework:** Laravel 10
- **Frontend:** 
  - Blade templates
  - Bootstrap 5
  - Vanilla JavaScript
  - Chart.js
- **Base de Datos:** MySQL
- **Diseño:** 
  - Glassmorphism UI
  - Responsive design
  - Dark theme

### Requisitos del Sistema

- PHP 8.1 o superior
- Composer
- Node.js y NPM
- MySQL

### Instalación

1. Clonar el repositorio
```bash
git clone https://github.com/imorlab/ilaravel.git
```

2. Instalar dependencias de PHP
```bash
composer install
```

3. Copiar el archivo de configuración
```bash
cp .env.example .env
```

4. Generar clave de aplicación
```bash
php artisan key:generate
```

5. Configurar la base de datos en el archivo .env

6. Ejecutar migraciones
```bash
php artisan migrate
```
